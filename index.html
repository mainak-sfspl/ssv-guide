Welcome! Sampurna Secure Vault (SSV) is our centralized, self-hosted secrets platform built for security, trust, and operational sanity. It keeps passwords, API keys, certificates, and automation credentials out of spreadsheets and source code — with MFA, audit logs, and role-based access.
Zero secrets in code Apps fetch time-bound secrets from Vault — no hardcoding, no accidental Git leaks.
MFA + Audit OTP enforced. Every view/edit/delete is logged. Accountability built-in.
